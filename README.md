# Academy-Course-CNC13010

Supporting Material for the Open Risk Academy course on measuring credit name concentrations 

<img src="https://www.openriskacademy.com/images/CNC13010/NameDiversification2.png" alt="Visual representation of diversifying name concentration in a portfolio" title="Name concentration measurement aims to provide objective metrics (indexes etc.) that help assess whether a given credit portfolio has too large concentration on a particular counterparty. This information can then be used to manage down the risk - diversify it away" role="presentation" class="course-thumbnail" align="center"  height="70" width="170" />

#### Summary:

This course is a DeepDive into measuring <i>name</i> concentrations in credit portfolios. 

#### Content:

It covers the following topics: 

<ul>
  <li>The measurement of credit name concentrations using basic indicators and indexes such as the concentration ratio and the HHI index</li>
  <li> The measurement of credit name concentrations using more advanced (risk based) indicators such as volatility and economic capital based indexes</li>
</ul>
